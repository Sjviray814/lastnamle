Welcome to Lastnamle, a game created by Stephen Viray and Chris Choi!

How to Play:

1. Try to guess which country the last name is most prevalent in 2. Use the hints to help you make your next guess. Continent: Green if you have guessed the correct continent, red if not. Region: Green if you have guessed the correct region within that continent (South Asia, Northern Europe etc.) First Letter: Green if you have guessed a country with the same first letter as the correct country # of Letters: Green if you have guessed a country with the same number of letters as the correct country 3. Keep guessing, try the unlimited mode if you want to continue playing!

About the Data:

- All of the data for this site was collected from Forebears. - The country associated with each name is decided by the country which has the most people of that last name, not based on the language of origin of the last name - The difficulties in the unlimited version are determined by the number of people in the 'answer country' that has that name. Here are the difficulties: Beginner: 1,000,000+ people in the #1 country have this last name Easy: 200,000+ people Normal: 50,000+ people Hard: Between 10,000 and 200,000 people Expert: Between 2,000 and 50,000 people Impossible: Between 0 and 10,000 people
